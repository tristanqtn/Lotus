Lotus
API request capture, inspect and replay tool for developers

How to Access Lotus:

  1. Open Developer Tools (F12 or Ctrl+Shift+I)
  2. Click on the Lotus tab in the DevTools panel (You may need to click the >> overflow menu to find it)
  3. If you don't see Lotus, close and reopen DevTools

Key Features:

  • Auto-Capture: Automatically logs all API requests
  • Filter: Search for specific requests
  • Copy as cURL: Export any request as cURL command
  • Modify & Resend: Change and replay requests
  • Theme Toggle: Switch between light and dark modes
  • Format Toggle: Switch between raw and pretty formatting

How To Use:

  • Request List: Shows all captured API requests in the left panel
  • Request Details: Click any request to view its headers and body
  • Response Details: View response status, headers, and body
  • Pretty/Raw: Toggle between formatted and raw data views
  • Modify: Click "Modify & Resend" to edit and send a new request
  • Filter: Type in the filter box to find specific requests

Tips & Tricks:

  • Use the filter to find requests by URL, method, or status code
  • JSON responses are automatically formatted for better readability
  • Modified requests appear at the top of the request list
  • Clear the request list to start fresh for a new debugging session
  • Easily test API endpoints by modifying captured requests
